# Urban Transition Boardroom — End-to-End Simulation

A guided executive project simulation covering Unit 3 and Unit 4 from city discovery through post-deployment assurance.

## What participants do

- investigate Al Noor City districts and pressure layers
- review evidence packs
- write a city diagnosis
- establish a sustainability baseline
- define strategic priorities and a business case
- draft technology requirements
- score technology options using a live SCALEC matrix
- make and reopen board decisions
- respond to delivery and cyber events
- manage a traceable risk register and matrix
- review generated KPIs and a final executive report

All progress is stored in the browser using localStorage.

## Deploy on Vercel

Set the Root Directory to `urban-transition-boardroom` and deploy with the default Next.js settings.

## Important

This version uses scenario calculations for learning. The dashboard clearly distinguishes sourced baselines from simulated decision impacts.
